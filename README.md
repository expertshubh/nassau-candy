# nassau-candy
Nassau Candy Distributor Factory Reallocation & Shipping Optimization System uses machine learning to predict shipping lead times and simulate factory-product reassignments across 10,194 historical orders. Three models were trained — Linear Regression, Random Forest, and Gradient Boosting — with Linear Regression selected as the best performer to power a scenario simulation engine. The project delivers a live Streamlit dashboard with Factory Simulator, What-If Analysis, Recommendations, and Risk & Impact modules to help leadership make data-driven factory assignment decisions.
